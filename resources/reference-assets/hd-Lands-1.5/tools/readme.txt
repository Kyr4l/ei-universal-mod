#####################
Программа предназначена для быстрой запаковки и распаковки Res архивов, а также конвертации reg<->ini.
Для конвертации reg<->ini необходимо, чтобы в папке с программой лежали файлы reg2ini.exe и ini2reg.exe.

Чтобы запаковать папку в архив, надо перетащить папку на ярлык программы. Аналогично, чтобы распаковать архив, надо перетащить его на ярлык программы.

Чтобы не перетаскивать, можно добавить программу в контекстное меню проводника (меню по правому клику мышки на файл/папку)
Для этого надо:
1) Открыть файл eipacker.reg блокнотом,
2) Найти строчку "C:\\Soft\\eipacker.exe" и заменить её на путь, куда вы сохранили eipacker.exe, при этом слэши (\\) обязаны быть двойными. Т.е. если у вас eipacker.exe лежит в папке c:\downloads\eipacker.exe, то итоговый путь будет c:\\downloads\\eipacker.exe
3) Запустить eipacker.reg двойным кликом мышки.

Программа работает так (аналогично EiEdit):
Если ей передали папку с именем folder_res, то программа её запакует в файл с именем folder.res
Аналогично, файл с именем folder.res распакется в папку с именем folder_res.
Точно так же: folder_mq -> folder.mq, folder.mq -> folder_mq
и т.д. с любыми другими расширениями.
