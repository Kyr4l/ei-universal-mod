1. Установка и настройка игры

Перед установкой мода требуется:
•	Установить игру 
•	Установить стартер в отдельную папку (см п. 2)
Запустите стартер
1.	Выберите рядом с пунктом «ИГРА» HD pack текущей версии. 
1.	Зайдите в настройки. Установите желаемые настройки графики, предварительно убрав галку с пункта «Фильтрация видео режимов» установите галку возле пункта «Дополнение» и нажмите кнопку «Дополнение…»
 2.	Ставим галку: Запускать игру без CD 
3.	Переходим на вкладку «Специальные» и устанавливаем галки возле 2х пунктов: «Включить поддержку консоли в MP» и «Камера Чейза» (остальное по желанию) 
4.	Жмем «Да» - «Да» и запускаем игру. В правом нижнем углу должна появится надпись
 «HD Lands (сборка; версия)» 
5.	Приятной игры!

ВНИМАНИЕ! Для корректного отображения текстур без замыливания при отдалении, настройте Анизотропную фильтрацию для игры. В случае с видеокартой от NVidia, данная настройка может быть доступна в Панели Управления nVidia. Общий для всех вариант - установить dgVoodoo 2, см. пункт 3.ж "Проблемы и их решение"."У кого проблемы с низким fps ..."

2. Скачивание и установка мода

Для ручной установки нужно:
1.	Установите игру (https://yadi.sk/d/6RVAIWQmnjWpsA). Пример: D:\Games\EvilI slands
2.	Установить стартер (https://yadi.sk/d/04YvbnAlp_MqNQ) в отдельную папку. Пример: D:\Games\EvilI slands\Starter
3.	Обновить addon.dll (https://yadi.sk/d/SqGCuFbRiIu5yQ) стартера. Пример: D:\Games\EvilI slands\Starter
4.	Распаковать мод (скачать, если есть новее: https://disk.yandex.ru/d/ubR9CVP6-2vl3w?w=1) HDpack(сборка; версия).7z в папку Mods в директории стартера. 
Пример D:\Games\EvilI slands\Starter\Mods\HDpack

3. Проблемы и их решение

  a) Игру нерекомендуется устанавливать на диск C:\ и в пути к игре не желательны русские буквы – используйте английские: D:\Games\EvilIslands\
  б) Скачайте файл mss32.dll и скинуть в папку с игрой D:\Games\EvilIslands\
https://yadi.sk/d/_u6tPKFqM8pGsw
  в) Выключить антивирус.
  г) Запустить игру от имени Администратора (правый клик по ярлыку запуска игры или стартера):
 
  д) Запуск игры в режиме совместимости с win98 (правый клик по ярлыку запуска игры или стартера – свойства – вкладка совместимость):
 
   е) Ингода не хватает прав для автоматического обновления плагина через интернет, поэтому стоит сделать это вручную:
Распаковать плагин (https://yadi.sk/d/jo1KQoh93PBsTr) в директорию игры. Пример: D:\Games\EvilI slands\
ж) У кого проблемы с низким fps - можно испрвить установкой dgVoodoo.
Экзешник и cfg вуду и дллки из папки MS/x86 кидаются в папку с game.exe; при игре через стартер это папка со стартером / Engine /.
В настройках dgVoodoo ставится general > Output API - DirectX 12, тогда не будет белых артефактов.
