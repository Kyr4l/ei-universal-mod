## MobExplorer
## (c) Demoth. 2014 
#####################
Небольшая утилита, которая позволяет просматривать и изменять содержимое .mob-файлов.
Имеет очень мощный редактор скриптов с возможностью проверить их корректность.
eisc_con.exe - консольная утилита для проверки синтаксиса скриптов.
ei_syntax.xml - содержит настройки для подсветки синтаксиса в редакторе.
script_refs.txt - текстовое описание скриптовых функций для подсказок.
Его можно и нужно редактировать. Для этого открыть его надо в Excel, ни в коем случае не используйте блокно для этого.
