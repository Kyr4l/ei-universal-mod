📘 MOB Scripts Editor - User Guide

Complete guide to the script editor features for Evil Islands

📂File Operations

Opening and Saving

ActionDescription
Open FileSupports *.mob (binary MOB files), *.txt (text files) and any other formats. When opening a MOB file, all objects, units, levers, traps, etc. are automatically parsed.
SaveSaves the file in its original format. For MOB files, recalculates all chunk sizes and logic before writing.
Save AsSaves the file under a new name.
Save CopyCreates a copy of the current file to a specified location (via selection dialog).
Drag & DropDrag files directly into the editor window.
Command Line LaunchIf a file path is passed as an argument at launch, the editor will automatically open it.

Backup System

FeatureDescription
Backup on SaveCreates a file with .bak extension in the same folder on every save.
Auto-backupsAutomatic backup creation at a configurable interval (default 5 minutes, can be configured, enabled or disabled).
Backup RotationStores up to 10 backups with numbers (.bak1, .bak2, ... .bak10). Old ones are automatically deleted.
Interval ConfigurationThe autosave interval can be changed via the Settings menu (from 1 minute to 24 hours).
Tip: All backup settings are saved in the Windows registry (HKEY_CURRENT_USER\Software\EIScriptEditor) and restored on the next launch.

✏️Text Editing

Basic Operations

HotkeyAction
Ctrl+ZUndo the last action
Ctrl+YRedo the undone action
Ctrl+CCopy selected text
Ctrl+XCut selected text
Ctrl+VPaste text from clipboard
Ctrl+ASelect all text

Text Selection

MethodDescription
MouseHold the left button and drag. When going beyond screen boundaries, it automatically scrolls.
Shift + ClickSelect from the current caret position to the click location.
Shift + ArrowsKeyboard selection by characters/lines.
Double ClickSelect the word under the cursor.
Home / EndGo to the beginning/end of the line (with Shift — select).

Auto-indent

When pressing Enter, the editor automatically carries over the indent from the previous line. The feature can be enabled/disabled via the menu.

🧭Navigation

Sidebar

On the left, there is a panel with two modes:

ModeDescription
Lines (Line Numbers)Shows line numbers. You can drag the right border to change the width.
Funcs (Function List)Shows all functions declared via DeclareScript in the current file. Clicking a function moves the caret to its declaration.
Tip: The sidebar width can be changed by dragging the right border with the mouse. The setting is remembered.

Find and Replace

HotkeyAction
Ctrl+FOpen the find/replace dialog
Ctrl+RReplace the next occurrence
F3Find the next occurrence

The search dialog supports:

  • Forward and backward search
  • Case sensitivity (optional)
  • Text replacement mode
  • Navigation through results using Enter and Esc

🎨Syntax Highlighting

The editor supports full color scheme customization via the syntax.ini file (in the EN\ folder).

Customizable Categories

CategoryDescription
KeywordsLanguage keywords (if, else, while, return, etc.)
TypesData types (float, string, object, group)
FunctionsFunction names (KillUnit, GetObject, etc.)
IdentifiersIdentifiers (variables, script names)
NumbersNumeric constants
StringsString constants in quotes
CommentsComments //
OperatorsOperators (+, -, *, /, =, brackets, etc.)

For each category, you can configure:

  • Main text color
  • Color when selected
  • Background color
  • Background color when selected

⚡Autocomplete

When typing a function name, a dropdown list with matching options automatically appears.

KeyAction
↑ / ↓Navigate through the list
Enter / TabInsert the selected function
EscapeClose the list without inserting
How it works: The list is filtered as you type. For example, when typing Kill, all functions starting with "Kill" will be shown (KillUnit, KillScript, etc.).

💡Parameter Hints

When typing an opening bracket ( after a function name, a hint with the function signature appears:

  • Function name
  • List of parameters with types
  • Function description (if enabled)

Function descriptions are loaded from the scripts.htm file (in the EN\ folder). Displaying descriptions can be enabled/disabled via the menu.

🛠️Tool Windows

Object Editor

A powerful editor for working with MOB files. Allows editing all types of objects:

  • Objects — static objects (buildings, rocks, trees)
  • Units — units (characters, enemies, NPCs)
  • Levers — levers and doors
  • Traps — traps
  • Torches — torches
  • Lights — light sources
  • Sounds — sound sources
  • Particles — particle sources

Object Editor capabilities:

FeatureDescription
Field EditingAll object fields are available for editing (ID, Name, Position, Rotation, etc.)
Unit Logic5 logic slots for each unit with AI behavior configuration
Guard PointsPatrol and guard points with actions
SortingSort objects by ID, Name, Type, Player or Position
Auto-applyAutomatic saving of changes when switching between objects
List ResizeThe object list width can be changed by dragging
Toast NotificationsPop-up save notifications

Helper Editor

An additional window with tabs for working with various aspects of the MOB file.

Ranges Window

Range editor for MOB files.

Diplomacy Window

A 32×32 diplomacy table for configuring relationships between groups:

  • Left click — toggle state (ally/neutral/enemy)
  • Right click — select cell
  • Alt + click — symmetrically change the reverse cell

⚙️Settings

All settings are available via the Settings menu and saved in the Windows registry:

SettingDescription
Auto-backup intervalInterval for automatic backup creation (from 1 minute to 24 hours)
Scroll speedSpeed of automatic scrolling when selecting beyond screen boundaries (in milliseconds)
File AssociationAssociate .mob files with this editor, MobSurgeon, or no association
Follow CaretAutomatic scrolling to follow the caret
Auto IndentAutomatic indent carry-over when pressing Enter
Show DescriptionsDisplay function descriptions in hints
File Association: When selecting "Associate with this editor", the editor is registered in the registry as the handler for .mob files. When selecting "Associate with MobSurgeon", the association with MobSurgeon is restored (if it was previously installed).

🚀Launch and Usage

Normal Launch

Simply run EIScriptEditor.exe with a double click.

Launch with File

Pass the file path as an argument:

EIScriptEditor.exe "C:\Games\EI\Maps\mymap.mob"

Drag & Drop

Drag the file directly into the editor window.

File Association

After configuring the association via the Settings menu, double-clicking any .mob file will automatically open it in the editor.

Important: Changing file association may require running the editor as administrator.
