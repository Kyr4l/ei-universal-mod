 Версия: 0.11.0.0                                       [26.10.2020 03:00]

[+] Улучшены некоторые скриптовые команды:
    1.  Улучшена функция GetUnitExp(unit: object) : float
        Добавлен опциональный аргумент для указания типа опыта:
        GetUnitExp(unit: object, exp_type: string) : float
        Доступные типы опыта:
        - "max" - максимальный полученный юнитом опыт.
            От него рассчитывается максимальное количество здоровья и маны.
            Используется по-умолчанию (когда тип не указан).
        - "spent" - опыт, потраченный юнитом на навыки/перки
        - "penalty" - размер штрафа за смерть (отрицательное число).
            Пока он не станет равным 0, здоровье и мана у игрока не будут расти.
        - "real" - опыт, отображаемый в магазине. Вычисляется как max + penalty
    2.  Новый опциональный аргумент для функции FloatToStr
        В дополнение к существующей функции доступен новый вариант:
        FloatToStr(num: float, digits: float) : string, где digits - число знаков после точки.
    3.  Улучшены команды Say и ConsoleStringb>
        Теперь они могут принимать сколько угодно строк, которые будут объединены в одну строку
        перед выводом в консоль/на экран.
[+] Добавлены новые скриптовые функции:
    1.  GiveUnitExp(unit: object, exp: float)
        Выдать exp опыта юниту unit.
    2.  GetDexterity(unit: object): float
        Возвращает ловкость юнита unit.
    3.  GetIntelligence(unit: object): float
        Возвращает разум юнита unit.
    4.  GetStrength(unit: object): float
        Возвращает силу юнита unit.
    5.  GetActions(unit: object): float
        Возвращает действия юнита unit с учётом эффектов заклинаний (бафов).
    6.  GetActionsBase(unit: object): float
        Возвращает действия юнита unit без учёта эффектов заклинаний (бафов).
    7.  GetSkill(unit: object, skill: string): float
        Возвращает навык skill юнита unit.
        Доступные навыки: "science", "melee", "archery", "elemental", "sense", "astral".
    8.  GetEncumbrance(unit: object): float
        Возвращает текущую нагрузку юнита unit.
    9.  GetMaxEncumbrance(unit: object): float
        Возвращает максимальную нагрузку юнита unit.
    10. GetPerk(unit: object, perk: string): float
        Возвращает уровень прокачки перка perk у юнита unit.
        Доступные перки: "sword", "axe", "dagger", "hammer", "bow", "crossbow", "fire",
          "lightning", "acid", "illusion", "divination", "enchantments", "healing",
          "domination", "infra", "health", "mana", "vitality", "spirit", "quickness",
          "lift", "quickness", "backstab", "strength", "dexterity", "intelligence".
    11. GetPrototype(unit: object): string
        Возвращает строку с именем прототипа юнита unit.
    12. GetRace(unit: object): string
        Возвращает строку с именем расы юнита unit.
    13. RandObj(grp: group): object
        Возвращает случайный объект из группы grp.
    14. GameOver()
        Завершить одиночную игру с сообщением о гибели главного героя.
    15. GameOver(title: string, message: string): object
        Завершить одиночную игру с указанными строками для заголовка окна и сообщения.
        Эти строки используются для поиска соответствующего файла с текстом в архиве texts.res.
        Например, текст для заголовка окна ищется в texts.res с именем файла "string _заголовок_".
[+] Добавлены новые консольные команды:
    1.  Команда nogui 1, которая позволяет отключить элементы интерфейса в игровой зоне.
        А также nogui 2 - которая в добавок к интерфейсу отключает так же курсор и nogui 0,
        которая включает интерфейс и курсор обратно.
    2.  Читерская команда drawall 1, которая включает отображение всех юнитов на карте.
        Полезна для разработчиков модов.
[+] Различные улучшения для возможности запуска ЗвА в качестве мода:
    1.  Добавлена поддержка загрузочных экранов из ЗвА.
    2.  Добавлена возможность загрузки библиотеки binkw32.dll из папки мода.
    3.  Добавлены скриптовые команды GameOver и GameOverMSG (см. выше).

[-] Исправлено несколько ошибок связанных с поддержкой широкоформатных дисплеев.
    Теперь поддержка широкоформатных дисплеев включена по-умолчанию.
[-] Исправлена ошибка скриптовой команды GiveUnitExp, из-за которой не отображалось
    количество полученного опыта.
[-] Исправлены различные ошибки оригинального движка, связанные с накоплением эффектов
    от заклинаний (бафов).
[-] Исправлена ошибка проверки навыка ловкости рук, необходимого для открытия замков,
    из-за которой навык ловкости рук требовался даже для замков с нулевым требованием к навыку.

[+] Добавлены новые скриптовые функции:
    1.  Random(min: float, max: float): float
        Возвращает случайное число в диапазоне от min до max. Новая функция является альтернативой
        оригинальной функции Random(max), которая будет продолжать работать, как и раньше.
    2.  RandStr(a: string, b: string, ...): string
        Возвращает случайным образом одну из переданных строк a, b, ...
    3.  SendCurrentParty(nPlayer: float)
        Отправить пользователю игрока с номером nPlayer его текущую партию. Другими словами,
        в сетевой игре, принудительно сохраняет на диск персонажа указанного игрока.
    4.  Loot(nPlayer: float, unit: object)
        Собрать труп мёртвого юнита unit в обоз игрока с номером nPlayer.
    5.  PlayerExists(nPlayer: float): float
        Возвращает 1, если игрок с номером nPlayer существует и 0 в противном случае.
    6.  Max(a: float, b: float, ...): float
        Возвращает максимальное число из аргументов a, b, ...
    7.  Min(a: float, b: float, ...): float
        Возвращает минимальное число из аргументов a, b, ...
    8.  Abs(num: float): float
        Возвращает абсолютное по модулю значение числа num.
    9.  Round(num: float): float
        Округляет число num до ближайшего целого числа. Например, 3.7 -> 4, 3.2 -> 3.
    10. Ceil(num: float)
        Округляет число num до ближайшего большего целого числа. Например, 3.7 -> 4.
    11. Floor(num: float)
        Округляет число num до ближайшего меньшего целого числа. Например, 3.7 -> 3.
    12. StrHas(str: string, substr: string): float
        Возвращает 1, если строка str содержит в себе подстроку substr и 0 в противном случае.
    13. StrMid(str: string, pos: float, len: float"): string
        Возвращает подстроку str с позиции pos и длиной len. Если позиция pos отрицательная, то
        считается позицией с конца строки. Если длина len равна 0, то функция возвращает подстроку
        с позиции pos и до конца строки str.
    14. StrLower(str: string): string
        Возвращает строку, состоящую из символов строки str в нижнем регистре.
    15. StrUpper(str: string): string
        Возвращает строку, состоящую из символов строки str в верхнем регистре.
[+] Улучшена консоль игры:
    1.  Добавлена поддержка хоткеев: ctrl+A, ctrl+C, ctrl+V, ctrl+X
    2.  Немного улучшена история команд, чтобы избежать её замусоривания одними и теми же командами

[*] Удалена скриптовая функций Eval в связи с тем, что от неё крайне
    мало пользы, но есть несколько потенциальных проблем. Возможно, в будущем появится её более полезный аналог

[-] Исправлена баг, из-за которой удаляются настройки клавиатуры при проверке скриптов
[-] Исправлена баг проверки минимальной версии аддона

    FastTeleport=1                    - Уменьшить минимальную длительность каста с 30 до 1 игровых секунд.
    DisableCrippledHL=1               - Отключить ломание конечностей.
    UnlockExperimentalSpell=1         - Разблокировать экспериментальные заклинания.
[+] Добавлены новые скриптовые функции:
    1.  Eval(script: string, count: float)
        Выполнить строку script, в качестве скрипта. В script используются одинарная кавычка
        вместо двойной для записи строк. Пример script: "ConsoleString('Hello!')".
    2.  GSSetPlayerVar(nPlayer: float, name: string, value: float)
        Установить значение переменной для конкретного игрока nPlayer. Имеет смысл только для
        сетевой игры, т.к. в одиночной игре она эквивалентна команде GSSetVar.
        В сетевой игре переменные, создаваемые командой GSSetVar общие для всех игроков.
    3.  GSGetPlayerVar(nPlayer: float, name: string) : float
        Возвращает значение переменной, установленной командой GSSetPlayerVar.
    4.  GiveExp(nPlayer: float, exp: float)
        Выдать опыт игроку с номером nPlayer. Если игрока с таким номером нет (например, -1),
        то опыт выдаётся всем игрокам в пропорциях аналогичным стандартной выдаче опыта
        при убийстве монстра.
    5.  StartCameraSave(path: string)
        Начать запись движений камеры в файл path.
    6.  StopCameraSave()
        Завершить запись движения камеры.
    7.  Say(message: string)
        Написать сообщение message в чат.
    8.  And(a: float, b: float, ...) : float
        Логическое "И". Возвращает 1, если все аргументы равны 1. Иначе возвращает 0.
    9.  Or(a: float, b: float, ...) : float
        Логическое "ИЛИ". Возвращает 1, если хотя бы один аргумент равен 1. Иначе возвращает 0.
[+] Из EI Plugin перенесено исправление номеров игроков, которое позволяет многим скриптам указывать
    отличное от 0 значение. Т.е. nPlayer в параметрах функций - реальный номер игрока.
    Чтобы получить номер игрока, используйте GetPlayerOfUnit. Важно учитывать, что этот номер
    может меняться в течении игры (например, при заходе/выходе игроков на карту).
[+] Так же из EI Plugin перенесены новые скриптовые функции:
    1.  GetPlayerOfUnit(unit: object) : float
        Возвращает номер плеера для юнита, если имеется. Для всех бесплеерных юнитов возвращает 0
    2.  GetUnitExp(unit: object) : float
        Возвращает общий опыт юнита, полученный за всё время игры
    3.  GetArmor(unit: object, part: float, id: float) : float
        Возвращает броню юнита.
        part - часть тела: 0 (голова), 1 (торс), 2 (л.рука), 3 (п.рука), 4 (л.нога), 5 (п.нога), -1 (общая броня)
        id - тип брони: 0 (impaling), 1 (slashing), 2 (crushing), 3 (termical), 4 (chemical), 5 (electrical), 6 (general)
    4.  GetObjectStringID(object) : string
        Возвращает ID объекта в виде строки
    5.  FloatToStr(float) : string
        Перевести число в строку
    6.  StrToFloat(string) : float
        Перевести строку в число
    7.  StrCat(string, string, ...) : string
        Объеденить две и более строк в одну
    8.  StrLen(string) : float
        Длина строки
    9.  Sin(float) : float
        Синус
    10. Cos(float) : float
        Косинус
    11. Pow(base : float, pow : float) : float
        Возвести число в степень (основание ^ степень)
    12. DistancePointPoint(x1: float, y1: float, x2: float, y2: float) : float
        Возвращает расстояние между точками
    13. NearestUnit(x: float, y: float, r: float, flags: float) : object
        Ближайший к точке (x,y) юнит. Поиск ведется в радиусе r
        Если flags = 1, то учитываются одушевленные объекты (люди, монстры и т.д.)
        Если flags = 2, то учитываются неодушевленные объекты (дома, деревья и т.д.)
        Если flags = 3, то учитывается все объекты
    14. UnitsNearPoint(x: float, y: float, r: float, flags: float) : group
        Получить список юнитов возле точки. Правила аргументов как для NearestUnit.
[+] Изменено несколько существующих скриптовых функций:

Основное
~~~~~~~~
[+] У пользователей WinXP, Win2K имеется проблема с переключением
    на русский язык во время игры.
    В этой версии библиотеки, Мы реализовали один из придуманных нами
    способов решения этой проблемы.
    Суть способа заключается в следующем: при запуске игры мы удаляем
    процесс ctfmon.exe, из-за которого, собственно говоря, и возникает эта
    проблема. После завершения игры мы вновь запускаем этот процесс.
