Evil Islands Starter v2 Beta

Description, changelog: see `docs\` folder.

Requirements:
Evil Islands v1.6
old EIStarter (v0.* - v1.46) (Optional)

Install:
Unpack to any convenient path.
If not installed old EIStarter: Run `Init Starter Registry.bat` for clone game`s Windows Registry branch (as Administrator!).
Run EIStarter.exe` or if not work: Run `EIStarter With Debug.bat` and see `starter.log`.
