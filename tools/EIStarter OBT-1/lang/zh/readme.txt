系統需求

基本需求
	Windows 95/98/2000，需安裝DirectX 7.0
	Pentium II 233 MHz
	64 Mb主記憶體
	AGP 3D畫面加速器
	影像卡上需有8MB的記憶體
	音效卡

要在你的電腦上面安裝遊戲，需要550MB的未壓縮空間。
除此之外，為了讓遊戲能夠穩定的執行，強烈建議你必須
在硬碟上面額外空出300MB的未壓縮空間。


建議配備
	Windows 95/98/2000，需安裝DirectX 7.0
	Pentium III 450 MHz
	128MB的主記憶體
	AGP 4x 3D畫面加速器
	影像卡上需有32MB的記憶體
	音效卡

注意：使用Windows 98作業系統，可以得到較好的效率。

Voodoo Graphics、ATI 3D Xpert、Intel i740、S3 Trio 3D。

建議使用晶片：nVidia GeForce 2、GeForce、TNT2、TNT; 
3Dfx Voodoo 5、Voodoo 4、Matrox Millennium G400、G200、
ATI Radeon、Rage Fury Maxx、Rage 128、S3 Savage 2000、
Savage 4、3DLabs Permedia 3、Power VR 3。

支援的晶片組：3Dfx Voodoo 3、Voodoo Banshee、Matrox 
Millennium G100、ATI Rage Pro、Intel i752、S3 Savage 
3D、SIS 620v、 3DLabs Permedia 2、 Power VR 2、Ticket to

式下進行遊戲。

	聲音
為了更正聲音方面的不正常現象（突發的雜音、叮叮
聲等等），我們建議你在開始程式的選單中，切換到
「安全聲音模式」。

	滑鼠
為了讓安裝在你電腦中的其他廠牌驅動程式，可以支援
滑鼠上面的滾輪，我們建議你安裝原廠的驅動程式，以
免滾輪無法在遊戲中控制視角的縮放。在這種情況下，
Windows 98和2000都會使用內建的驅動程式。

	同時啟用其他程式
為了讓遊戲能夠穩定的執行，我們建議不要使用
Alt+Tab的組合鍵切換到其他程式之中。除此之外
，我們也建議你在執行遊戲之前關閉所有的應用
程式（尤其是微軟的Outlook）。

注意：為了加速遊戲從硬碟讀取的時間，我們建議你在遊
戲完整安裝完畢之後，執行一次磁碟重組程式把檔案整理
一下。

自動播放程式選項
這個部份的選單可以讓你變更預設的項目，包括畫面
品質和遊戲速度（你必須十分清楚你想要達成的效果
）。想要在變更選項之前，查看預設的參數值請點選
「自動偵測」的按鈕。

	顯示卡驅動程式
選擇你顯示卡所用的驅動程式。如果你的電腦上面安裝
了好幾塊顯示卡，你可以選擇要使用那一塊來進行遊戲
。
注意：遊戲使用的驅動程式只會透過Direct3D介面；並不
支援Glide或OpenGL的API。

	解析度和顏色
選擇遊戲進行時所用的畫面模式。我們建議你至少選用
800x600x16的數值（表示畫面為800x600，顯示的顏色為
16位元的色素）。如果你的顯示卡十分低階，你可以選
擇640x480的模式。

	全螢幕
如果你點選這個地方，遊戲就會以全螢幕的方式
（預設值）來進行；否則就會在視窗模式中進行
遊戲（並不是所有的驅動程式都支援這個模式）。

	Mip貼圖
點選這個地方，你就可以從攝影機中看到更真實的
材質，但是會耗用更多的記憶體。如果你取消這個
功能，只會使用一種材質，而且有些地方可能會閃
爍。

	遞色
點選這個地方你就可以打開遞色模式。這個功能會
加強在16位元下的畫面品質，但是整體的效率可能
會稍微下降。

	拉長影像
點選這個地方，你就可以在全螢幕下觀看動畫。在
某些特定的顯示卡上，打開這個選項時會導致播放
動畫時十分緩慢。

	獨立游標
可以讓游標的移動更為順暢，但是降低了影像的表
現效率。

	濾鏡 
「Point」選項可以確保最快的速度（但是品質最差）；
「Bilinear」選項是很好的折衷；而「Trilinear」選擇可以提
供最佳的畫面（但是速度最慢）。

	打光頻率
選擇光線照在固定物品上面的頻率，「經常」的設定
可以提供最佳的畫面真實度；「中等」則提供折衷的
選項；「稀少」數值提供了最快的速度。

地形品質–選擇地形精細程度的品質。「高」可以呈
現出最好的品質，「中」表示折衷的選項，「低」則
可以確保遊戲進行的速度最快。

	材質品質
選擇材質的品質。「高」可以呈現出最好的品質，
「中」表示折衷的選項，「低」則使用了最少的影
像卡記憶體。

	安全聲音
打開這個功能可以切換到另一套音效系統，會導致
CPU的負載加重（預設值是關閉的）。如果你的音
質聽起來很差，可以把這個功能打開。
