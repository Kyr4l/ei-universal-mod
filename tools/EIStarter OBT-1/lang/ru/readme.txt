1. Системные требования

Минимальная конфигурация:
Windows 95/98/2000, DirectX 7.0
Pentium II 233 МГц
64 Мб ОЗУ
Графический 3D-ускоритель AGP
8 Мб видео-ОЗУ
Звуковая карта
Мышь

Для установки игры на ваш компьютер необходимо иметь 550 Мб 
некомпрессированного свободного места на жестком диске. Для
устойчивой работы игры после инсталляции дополнительно необ-
ходимо иметь 300 Мб некомпрессированного свободного места на
жестком диске.

Рекомендуемая конфигурация:
Windows 95/98/2000, DirectX 7.0
Pentium III 450 МГц
128 Мб ОЗУ
Графический 3D-ускоритель AGP 4x
32 Мб видео-ОЗУ
Звуковая карта
Мышь

Примечание: при наличии на вашем компьютере операционных
систем  Windows 95/98 и Windows 2000 наиболее предпочти-
тельно использовать Windows 95/98.

3. Возможные технические проблемы и их решение

	Видео
Рекомендуемые видеоадаптеры: nVidia GeForce 2, GeForce, TNT2,
TNT; 3Dfx Voodoo 5, Voodoo 4; Matrox Millennium G400, G200; 
ATI Radeon, Rage Fury Maxx, Rage 128; S3 Savage 2000, Savage 4;
3DLabs Permedia 3; Power VR 3.

Поддерживаемые игрой видеоадаптеры: 3Dfx Voodoo 3, Voodoo
Banshee; Matrox Millennium G100; ATI Rage Pro; Intel i752;
S3 Savage 3D; SIS 620v; 3DLabs Permedia 2; Power VR 2;
Ticket to Ride; Trident Blade 3D.

Игра не поддерживает следующие видеоадаптеры:
3Dfx Voodoo 2, Voodoo Graphics; ATI 3D Xpert; Intel i740;
S3 Trio 3D.

Примечание: на видеоадаптерах 3Dfx Voodoo Banshee и Matrox 
Millennium G400 рекомендуется запускать игру в оконном режиме.
Для этого в настройках программы автозапуска отключите опцию 
"Полноэкранное видео".

Примечание: некоторые из перечисленных адаптеров могут
искажать цвет, тектстуры, видд курсора, работать устойчиво
только в оконном режиме.
В этом случае мы рекомендуем использовать драйвера, которые
находятся на первом диске в каталоге \Drivers.

	Звук
В случае проявления каких-либо искажений звука рекомендуется
включить параметр "Безопасный звук" в установках программы
автозапуска.

	Мышь
Если у вас установлены драйверы мыши с колесиком прокрутки 
(wheel mouse) от фирм-производителей и не действует функция 
приближения/удаления камеры с помощью колесика, то рекоменду-
ем перед запуском выключить или деинсталлировать оригинальный
драйвер. Тогда Windows 98 или 2000 станет использовать драйвер,
встроенный в систему.

	Совместный запуск приложений
Для устойчивой работы игры настоятельно рекомендуется
не переключаться на другие приложения с помощью комбинации
клавиш Alt-Tab. Кроме того, перед запуском игры желательно
закрыть все другие приложения чтобы максимально освободить
системную память.

Примечания: для ускорения процесса загрузки данных игры с
жесткого диска рекомендуется после установки игры осуществить
дефрагментацию жесткого диска с помощью системной утилиты
Disk Defragmentor (defrag.exe).

4. Настройки в программе автозапуска

Настройки позволяют изменить установленные по умолчанию пара-
метры, которые влияют на качество изображения и скорость 
работы. Для определения параметров по умолчанию при изменении
конфигурации вашей машины нажмите кнопку «Автоопределение».

	Primary Display Driver (Видеодрайвер)
Выбор видеоадаптера. Если на компьютере установлены несколько
видеоадаптеров, то можно указать, какой из них должна исполь-
зовать игра.
Примечание: игра работает только через интерфейс Direct3D. 
API Glide и OpenGL не поддерживаются.

	Выбор разрешения и глубины цвета
Выбор видеорежима, в котором запустится игра.
Рекомендуется не ниже 800x600x16 (число точек по ширине и 
высоте экрана, число бит цвета на точку).
При очень низких параметрах быстродействия видеоадаптера 
допустимо использовать режим 640x480.

	Полноэкранный режим
Если этот параметр отмечен галочкой, игра запускается в 
полноэкранном режиме (по умолчанию). В противном случае - 
в оконном режиме (поддерживается не всеми видеоадаптерами).

	Мип-маппинг
Выводит реалистичные текстуры на больших расстояниях от 
камеры, при этом расходуется дополнительная память.
При выключении используется одна и та же текстура, может 
наблюдаться «рябь».

	Псевдотонирование
Включение псевдотонирования. Улучшает качество изображения 
в 16-битном режиме цветности. Возможно небольшое снижение 
производительности.

	Полноэкранное видео
Запуск видеороликов в полноэкранном режиме. Включение
параметра на некоторых видеоадаптерах может привести к замет-
ному замедлению при просмотре роликов.

	Быстрый курсор
При низкой производительности делает движения курсора более 
плавными, но понижает общее быстродействие видеосистемы 
(в Windows 9x).

	Фильтрация
Точечная даёт максимальную скорость (но с наихудшим качеством),
Билинейная - компромиссный вариант, Трилинейная - лучшее 
качество изображения (но при существенном снижении скорости).

	Скорость переосвещения
Частота переосвещения статических объектов. Частая даёт 
наиболее реалистическое изображение, Нормальная -
компромиссный вариант, Редкая - максимальную скорость.

	Детализация ландшафта
Дистанция переключения уровней детализации. Высокая даёт 
лучшее качество изображения, Средняя - компромиссный вариант,
Низкая - максимальную скорость.

	Качество текстур
Высокое даёт лучшее качество изображения, Среднее - 
компромиссный вариант, Низкое - минимальное
использование оперативной памяти и видеопамяти.

	Безопасный звук
Использование альтернативных механизмов проигрывания звука,
при этом создаётся дополнительная загрузка процессора (по 
умолчанию выключено). Включите эту установку, если при 
проигрывании звука слышны явные искажения и помехи.

5. Сетевая игра

	Технология
Сетевая игра реализована по технологии клиент-сервер и 
позволяет организовать совместную игру как по локальной сети, 
так и через Internet. Обмен информацией происходит по
протоколу UDP через порт 8888.
Сервера, находящиеся в локальной сети клиента, обнаруживаются
по широковещательному запросу и сразу видны в окне подключения.
Для соединения по Internet адрес сервера должен быть записан в
адресную книгу игры (в виде доменного адреса без префикса или
IP-адреса). При входе в окно подключения клиент периодически
опрашивает адреса, записанные в книгу (сервера из книги 
выделяются цветом). Как только сервер обнаружен, в строке вместо
знака вопроса появляется имя сервера, количество игроков, название
острова и время отклика. В правой части окна подключения выводятся
параметры сервера, на который указывает маркер.

Имя сервера:	[название, которое дано серверу, по умолчанию
		совпадает с именем игрока]

Игроков:	[два числа через дробь: текущее и максимальное
		количество игроков]

База:		[название острова, на котором происходит игра]

Задание:	[название взятого задания или пробел, если 
		задание	пока не взято]

IP:		[IP-адрес сервера и номер порта]

Пинг:		[время отклика на ping в миллисекундах, при 
		отсутствии связи 9999]

	Требования к линии связи
Для игры по локальной сети достаточно иметь соединение по 
Ethernet со скоростью 10 Мбит/с, дополнительных настроек 
не требуется. При игре по модему рекомендуется иметь скорость 
соединения не ниже 28,8 Кбит/с для клиента. Полоса канала для 
сервера должна быть больше, пропорционально числу подклю-
ченных игроков. При низкой ширине полосы скорость игры будет 
замедляться, особенно при наличии большого количества 
персонажей игроков и монстров на игровом поле. Для ограниче-
ния сверху полосы на одного клиента используется консольная 
команда на сервере:

rate 		[число байт/с на канал, пределы 500...10000]

По умолчанию в игре установлено rate 2500, что соответствует
одному подключенному клиенту при скорости соединения 
28,8 Кбит/с.

Дополнительные консольные команды:
Для ввода команд с консоли нажмите клавишу <~>, для закрытия
окна консоли нажмите <~> повторно.

kick		[имя_игрока] - отключить игрока с данным именем от
		сервера;

ban		[имя_игрока] - отключить игрока с данным именем и 
		запретить ему вход на сервер до конца
		сеанса игры;

fps on	или fps 1 - выводить на экран частоту смены кадров;

fps off	или fps 0 - отключить вывод частоты смены кадров;

quit	выход из игры в Windows.

В режиме многогопользовательской игры в процессе выполнения
задания, т.е. на игровой зоне, ставить паузу может только 
игрок, стартовавший Сервер.
